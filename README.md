# Logger
## log封装

### 使用方法

module 下添加

     compile 'com.yanyi.benyanyi:loglib:1.0.1'

<br/>

    可设置是否打印JLog.init(boolean),默认debug版本apk打印，release版本apk不打印
    
<br/>
